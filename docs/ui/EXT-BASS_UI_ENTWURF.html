Implementierungs-Mapping (für Claude Code)

BASS-MODEpara3_bass_mode(p,on) — Controller-Setting (kein Param-Trichter). AUS ⇒ alle Teilblöcke neutral ⇒ bitidentisch.
FAT PRESETpara3_bass_preset(p) — setzt kohärente Konfig NUR über die u. g. Setter/Params (kein Sonderpfad).
Waveform/Oszpara3_osc_wave(p,osc,wave) 0=Saw (Default = bitidentisch), 1=Pulse · bandlimitiert, Alias ≤ −74 dBc.
Pulse WidthParam BassPulseWidth via para3_set_param · Default 0.5 · wirkt nur bei Pulse.
PWM Depth + SrcParam BassPwmDepth (Default 0) · Quelle para3_bass_pwm_src (LFO/EG/Drift).
Bass SpreadParam BassSpread (Default 0) · additiv auf E2.1-ds; detune_ bleibt unberührt.
Drift Rate/DepthParams BassDriftRate/BassDriftDepth (Depth Default 0) · seedbar para3_bass_drift_seed.
Sub Level/OctParam BassSubLevel (Default 0 ⇒ mix+=0 exakt) · Oktave −12/−24 Controller-Setting.
Stack/Monopara3_bass_stack(p,on) — neuer Allocator-Modus; übersteuert Voice-Mode; AUS ⇒ Modi unverändert.
Character-MeterRead-only Indikator (Schwebung/Spektralfülle), kein Treue-Score.
Trichter-Params (Knöpfe): PW · PWM · Spread · Drift · Sub — setParamNorm, nur im EXT-Panel, NICHT in Keys-KNOB_PARAM Controller-Hooks (diskret): Mode · Preset · Waveform · Stack · Quellen/Oktave
Disziplin (aus dem Runbook): separates EXT-Panel, kein Umbau der bestehenden Oberfläche · Default AUS ⇒ bitidentisch (Haupt-Akzeptanz) · bandlimitiert/klickfrei, gemessen · Defaults sind // EXT-BASS DEFAULT, nicht CALIB(E8) · Playwright-Stories US-BASS-COLD/ENABLE/NOCLICK auf dem VPS grün.
